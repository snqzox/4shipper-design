# Foundations

The primitives beneath the components: **color**, **typography**, **spacing**, **radius/elevation**,
and **iconography**. Document each as its own file (e.g. `color.md`, `typography.md`) once the real
tokens are confirmed from the UI Kit. Style/variable data is pulled into `data/styles.json` and
`data/variables.json`; this folder is the human-readable explanation of how to use them.
