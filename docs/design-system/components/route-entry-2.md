## Description

Route Entry 2 is a supplementary route waypoint component that provides additional size/density variants not covered by the primary `Route Entry`. It is used in contexts where the route visualisation requires a more compact node indicator, such as nested route summaries or auxiliary route displays.

## Usage rules

- Use `Route Entry 2` only when `Route Entry` does not fit the required density or node style.
- Maintain visual consistency within a single route display — do not mix `Route Entry` and `Route Entry 2` node styles arbitrarily.
