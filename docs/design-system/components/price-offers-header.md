# Price offers header

**Component set key:** `e2e34b6576f057ce30584b5a9e64d5dd63bbc09a`
**Figma page:** `Tenders - Carrier`
**Variants in set:** 2

## Description

Price offers header is the header component for the carrier's price offer submission view. It displays context about the tender (route summary, deadline) and provides the primary call-to-action for the carrier to submit their price offer. Use it at the top of the carrier tender detail / offer submission page.

## Variants / Properties

### Property 1 axis
| Variant | Description |
|---|---|
| `Default` | Standard offer header — tender is open for submissions. |
| `Variant2` | Alternative state — verify in Figma (e.g., after a price offer has been submitted). |

## Usage rules

- Use only in the carrier-facing price offer submission view.
- Show the deadline prominently — carriers need to know when the submission window closes.
