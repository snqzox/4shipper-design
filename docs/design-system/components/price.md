## Description

Price is the composite price display component used when additional context is needed alongside the monetary value — for example, showing a base price together with a per-kilometre rate or currency breakdown. Use `Price/Basic` for simple value display; use `Price` when the full contextual price block is required.

## Variants / Properties

### Property 1 axis
| Variant | Description |
|---|---|
| `Default` | Standard full-context price block. |

## Related components

- `Price/Basic` — simplified size-variant price label for standalone value display.

## Usage rules

- Use `Price` in carrier offer cards and tender price summaries where multiple price dimensions need to be shown together.
- Use `Price/Basic` for single-value price display in table cells or compact UI.
