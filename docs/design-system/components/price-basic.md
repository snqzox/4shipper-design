# Price/Basic

**Component set key:** `d2333632e60796c68093f6afc645c7280656bd3c`
**Figma page:** `⚛️ Price`
**Variants in set:** 4

## Description

Price/Basic is a typographic display component for rendering a monetary value. It comes in four sizes to fit different layout contexts — from a prominent headline price to a compact inline value. Always use this component for price display to ensure consistent number formatting and currency presentation across the product.

## Variants / Properties

### Property 1 axis (size)
| Size | When to use |
|---|---|
| `XS` | Smallest — inline within dense table cells or metadata rows. |
| `SM` | Small — secondary price display alongside a primary price. |
| `MD` | Medium/default — standard card or summary use. |
| `LG` | Large — prominent hero price, e.g., the winning offer price on a tender. |

## Related components

- `Price` — the full price component with currency and per-unit breakdown.

## Usage rules

- Always include the currency symbol or code alongside the numeric value.
- Use `LG` for the primary price on a screen; use `SM` or `XS` for supplementary or comparative prices.
- Do not use plain text for price values — always use this component for correct formatting.
