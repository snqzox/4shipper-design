# Chat list entry

**Component set key:** `b6f57353af7024a9ff59b6aca66fe45b2546a1bd`
**Figma page:** `Tenders - Shipper`
**Variants in set:** 2

## Description

Chat list entry is the row component used in the Q&A or messaging panel of the tender detail view. It renders a single message or conversation entry — showing the sender identity, message preview, and timestamp. Use it to build the chat/Q&A thread list within a tender.

## Variants / Properties

### Property 1 axis
| Variant | Description |
|---|---|
| `Default` | Standard chat message row. |
| `Variant2` | Alternative treatment — verify in Figma (e.g., unread or highlighted message state). |

## Usage rules

- Display messages in chronological order (oldest first).
- Distinguish visually between shipper-authored and carrier-authored messages.
- Use only within the Q&A / messaging panel of a tender detail page.
