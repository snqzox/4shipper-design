# Assets (Icons & Helper Components)

**Figma page:** `🧩 Assets`
**Components in set:** 188 (ungrouped — no named component set)

## Description

The Assets page contains the full icon library and a collection of internal helper/template frames that do not belong to a named component set. The icons use FontAwesome-style naming (e.g., `truck`, `location-dot`, `calendar`, `ellipsis`, `trash`) and cover the full range of actions, objects, and status symbols used throughout the 4Shipper / Transportly product. The helper frames include pre-built content blocks (e.g., `Form Header`, `Content Header`, `Page/Footer`) used to speed up page composition.

## Icon naming convention

Icons are named using kebab-case descriptive names that match FontAwesome 6 naming (e.g., `arrow-right`, `circle-check`, `pen`, `magnifying-glass`). When inserting an icon, select the component by its exact icon name from the Assets panel.

## Helper frames (internal)

Helper frames on this page include `Form Header`, `Content Header`, `Page/Footer`, `Logo`, `Logo Small`, `Mail Footer`, and others. These are building blocks used inside page-level designs — do not use them as standalone atoms.

## Usage rules

- Always use icons from the Assets library; do not import external icon libraries or use raw SVGs.
- Pair icons with a visible label or tooltip wherever space allows for accessibility.
- Do not resize icon components — they are designed at a fixed size; scale by wrapping them in a frame if needed.
