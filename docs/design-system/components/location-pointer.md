# Location Pointer

**Component set key:** `930af1ca0440b03a904cc6ea7d486d615f5e4477`
**Figma page:** `⚛️ Location Pointer`
**Variants in set:** 2

## Description

Location Pointer is a map-pin marker component used on embedded map views to indicate a specific geographic location — for example, a pickup point, delivery address, or warehouse. Use it wherever a map surface is rendered to mark waypoints, depots, or positions of interest.

## Variants / Properties

### Property 1 axis
| Variant | Description |
|---|---|
| `Default` | Standard pin marker — single location, no additional state. |
| `Variant2` | Alternative pin style — check Figma for the specific treatment (e.g., selected or active state). |

## Usage rules

- Use Location Pointer only on map surfaces, not in list or card contexts.
- For route sequences, pair multiple Location Pointer instances with connecting lines to form a complete route visualisation.
