# Tender Header - Shipper

**Component set key:** `5c8825fd2d0c165dbb4fd4bb95efb3b3a78edd33`
**Figma page:** `Tenders - Shipper`
**Variants in set:** 2

## Description

Tender Header - Shipper is the top-of-page header for the tender detail view in the shipper flow. It displays the tender title, status badge, key dates, and primary actions (e.g., "Edit", "Cancel tender"). Use it at the top of every tender detail page.

## Variants / Properties

### Property 1 axis
| Variant | Description |
|---|---|
| `Default` | Standard tender header. |
| `Variant2` | Alternative layout — verify in Figma for the current treatment (e.g., a different status configuration). |

## Usage rules

- Use only in the shipper-facing tender detail view.
- Actions in the header must reflect the permissions available to the current user at the current tender status.
