# Progress Bar/Bar

**Component set key:** `d6b611082102b96edbde92ca153d7b3e4c1d4288`
**Figma page:** `⚛️ Progress Bar`

## Description

Progress Bar is a horizontal track-and-fill indicator that communicates the completion percentage of a process or the current value within a range. Use it for tasks with a measurable, determinate progress value — for example, upload completion, onboarding step progress, or cargo fill level.

## Variants / Properties

### Size axis
| Size | Description |
|---|---|
| `Default` | Standard height — use in most contexts. |
| `SM` | Smaller height — use inside compact cards or table cells where vertical space is limited. |

## Usage rules

- Only use Progress Bar for determinate progress (a known percentage). For indeterminate loading states, use a spinner or skeleton instead.
- Always accompany the bar with a numeric label or descriptive text so the value is accessible.
- Use `SM` size only in dense layouts where `Default` would dominate too much vertical space.
