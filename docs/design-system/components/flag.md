## Description

Flag is a collection of country and territory flag icons keyed by ISO 3166-1 alpha-2 code (e.g., `pl`, `de`, `fr`). Additionally, a small number of regional/supranational flags are included (`eu`, `un`, `arab`, `asean`, `cefta`, `eac`). Use Flag to visually represent nationality, origin/destination country, or carrier country of registration.

## Variants / Properties

### Property 1 axis

The single property axis holds the ISO 3166-1 alpha-2 country code (lowercase). Examples:

| Value | Country / Region |
|---|---|
| `pl` | Poland |
| `de` | Germany |
| `fr` | France |
| `gb` | United Kingdom |
| `eu` | European Union |
| `un` | United Nations |
| `xx` | Unknown / no flag |

271 flag variants are available covering countries worldwide.

## Usage rules

- Always use lowercase ISO codes to select the correct flag variant.
- Use `xx` for cases where the country is unknown or not applicable.
- Pair Flag with the country name in text when used in tables or lists — do not rely on the flag alone for identification (accessibility).
- Do not scale Flag icons to sizes larger than the intended icon size — they are not illustrations.
