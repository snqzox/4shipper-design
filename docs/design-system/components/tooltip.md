# Tooltip

**Figma page:** `⚛️ Tooltip`
**Variants in set:** 4

## Description

Tooltip is a non-interactive contextual hint that appears on hover (or focus for accessibility) to clarify the purpose of an element — typically an icon button, a truncated label, or a form field that needs extra explanation. It disappears when the user moves focus away.

## Variants / Properties

### Position axis
| Position | Description |
|---|---|
| `Top` | Tooltip appears above the trigger element. Default preference. |
| `Bottom` | Tooltip appears below the trigger element. |
| `Left` | Tooltip appears to the left of the trigger element. |
| `Right` | Tooltip appears to the right of the trigger element. |

## Usage rules

- Use Tooltip only for supplementary information — never for information essential to completing a task.
- Keep tooltip text concise (one short sentence maximum).
- Choose the `Position` that keeps the tooltip within the viewport and does not obscure nearby interactive elements.
- Do not put interactive elements (links, buttons) inside a Tooltip.
